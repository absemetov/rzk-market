# rzk-market-ua

## Build Setup

```bash
# projects list
$ yarn pl

# serve frontend
$ yarn --cwd functions/ parcel-s

# serve functions, hosting
$ yarn s

# build for production and launch server
$ yarn --cwd functions/ parcel-b
$ yarn deploy-site-*
$ yarn deploy-bot-*
$ yarn deploy-triggers-*
```
